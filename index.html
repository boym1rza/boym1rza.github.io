O'zgaruvchilar

O'zgaruvchilar ma'lumotlarni saqlash uchun ishlatiladi. Python'da o'zgaruvchi e'lon qilish uchun tur ko'rsatish shart emas.

x = 5
y = "Matn"
print(x)
print(y)

Ma'lumot turlari

Asosiy ma'lumot turlari:

  • int - butun sonlar (masalan: 10)
  • float - haqiqiy sonlar (masalan: 3.14)
  • str - matn (masalan: "Hello")
  • bool - mantiqiy qiymatlar (True/False)
  • list - ro'yxat
  • dict - lug'at
son = 42  # int
haqiqiy = 3.14  # float
matn = "Python"  # str
rost = True  # bool

Shart operatorlari (if-else)

Shartlarni tekshirish uchun ishlatiladi.

a = 10
if a > 5:
    print("a katta 5 dan")
elif a == 5:
    print("a teng 5 ga")
else:
    print("a kichik 5 dan")

Tsikllar

for tsikli:

for i in range(5):
    print(i)

while tsikli:

i = 0
while i < 5:
    print(i)
    i += 1

Funksiyalar

Funksiyalarni def kalit so'zi bilan yaratiladi.

def salom_ayt(ism):
    print(f"Salom, {ism}!")

salom_ayt("Ali")
